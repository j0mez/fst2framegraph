# Citation and attribution

If you use `fst2framegraph`, cite the software using `CITATION.cff`.

This package builds on external resources. Depending on your workflow, also cite:

- Frame Semantic Transformer: David Chanin. 2023. *Open-source Frame Semantic Parsing*.
- FrameNet: Collin F. Baker, Charles J. Fillmore, and John B. Lowe. 1998. *The Berkeley FrameNet Project*. DOI: 10.3115/980845.980860.
- FrameBase: Jacobo Rouces, Gerard de Melo, and Katja Hose. 2017. *FrameBase: Enabling integration of heterogeneous knowledge*. DOI: 10.3233/SW-170279.

FrameBase data is not bundled in this repository. Users download or register FrameBase files
separately, and those files remain under their own licence. See `THIRD_PARTY_NOTICES.md`.
